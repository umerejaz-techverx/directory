## Pre-requsites

    Python > 3.6
    Django >= 4

## Clone Repository

    git clone <url>
    cd <path to dir>

## Setup
    pip install -r requirements.txt 
    python manage.py makemigrations
    python manage.py migrate
    python create_account.py
    python manage.py runserver
